UI 动画卡顿很多时候不是“JS 不够快”，而是你不小心触发了昂贵的渲染路径。

## 一条常见的误解

“用了 transform 就一定快”并不完全对。  
关键还是看它会不会触发：

- layout
- paint
- composite

## 最小记忆法

把 GPU pipeline 想成一条流水线：

1. 顶点处理（Vertex）
2. 裁剪/组装（Assembly）
3. 光栅化（Raster）
4. 片元着色（Fragment）
5. 混合/输出（Blend/Output）

当你知道每一步在做什么，你就知道该避免什么、该拥抱什么。
